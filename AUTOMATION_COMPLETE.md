## How It Works

### Daily Workflow

```
9:00 AM UTC - Cron triggers OpenClaw agent
    ↓
Agent receives message: "Generate 10 new leads and email them 
to noctil.agent@agentmail.to with company and profile summaries"
    ↓
Agent uses lead-gen-pro skill to:
  1. Search for target companies (e.g., "SaaS companies founded 2025")
  2. Use agent-browser to visit company websites
  3. Extract contact information (name, title, email, LinkedIn, phone)
  4. Generate company summary (what they do, products, funding, etc.)
  5. Generate profile summary (background, experience, achievements)
  6. Check for duplicates against sent leads database
    ↓
Agent uses AgentMail skill to:
  7. Format each lead as a professional email
  8. Send to noctil.agent@agentmail.to
  9. Track sent leads to avoid duplicates
    ↓
Agent sends summary notification via Telegram
    ↓
Process complete - waits for next day
```
