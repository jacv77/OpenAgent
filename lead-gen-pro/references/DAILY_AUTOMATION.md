# Daily Lead Generation Automation

## Overview

Automatically generate 10 leads every day and email them to noctil.agent@agentmail.to with company summaries and profile information.

## Configuration

### Cron Job Setup

The daily lead generation is configured in `~/.openclaw/cron/jobs.json`:

```json
{
  "jobs": [
    {
      "id": "daily-lead-gen",
      "enabled": true,
      "schedule": "0 9 * * *",
      "description": "Generate 10 leads daily and email to noctil.agent@agentmail.to",
      "command": "python3 ~/.openclaw/skills/lead-gen-pro/scripts/daily_lead_gen.py",
      "timezone": "UTC",
      "notify": true,
      "notifyChannel": "telegram"
    }
  ]
}
```

### Schedule Format

The cron schedule `0 9 * * *` means:
- **0**: Minute (0 = top of the hour)
- **9**: Hour (9 = 9 AM)
- **\***: Day of month (every day)
- **\***: Month (every month)
- **\***: Day of week (every day)

**Result**: Runs every day at 9:00 AM UTC

### Customizing the Schedule

```bash
# Every day at 9 AM
0 9 * * *

# Every day at 6 PM
0 18 * * *

# Every weekday (Mon-Fri) at 9 AM
0 9 * * 1-5

# Twice daily (9 AM and 6 PM)
0 9,18 * * *

# Every 12 hours
0 */12 * * *
```

## Workflow

### 1. Search Discovery
```
- Search for target companies (e.g., "SaaS companies founded 2025")
- Extract company websites and basic information
- Filter out previously contacted companies
```

### 2. Data Extraction
```
For each company:
  - Visit company website
  - Navigate to About/Team pages
  - Extract contact information
  - Get company description
  - Find LinkedIn profiles
```

### 3. Summary Generation
```
Company Summary:
  - What the company does
  - Products/services offered
  - Recent news or funding
  - Company size and location

Profile Summary:
  - Contact person's role
  - Professional background
  - Years of experience
  - Relevant achievements
```

### 4. Email Delivery
```
For each lead:
  - Format email with company and profile summaries
  - Send to noctil.agent@agentmail.to
  - Track sent leads to avoid duplicates
  - Log success/failure
```

### Duplicate Detection

Leads are considered duplicates if:
- Company website matches a previously sent lead
- Contact email matches a previously sent lead
- Company name is very similar (fuzzy match)

## Error Handling

### Common Issues

1. **No New Leads Found**
   - Expands search criteria
   - Tries alternative search queries
   - Sends notification to review search terms

2. **Email Delivery Failure**
   - Retries up to 3 times with exponential backoff
   - Logs failed leads for manual review
   - Sends error notification

3. **Website Access Issues**
   - Skips inaccessible websites
   - Marks lead as "Incomplete"
   - Continues with remaining leads

4. **Rate Limiting**
   - Adds delays between requests
   - Respects robots.txt
   - Spreads processing over time

### Run Immediately


## Performance

### Expected Timing

- Search Discovery: 2-3 minutes
- Data Extraction: 5-8 minutes (10 leads)
- Summary Generation: 2-3 minutes
- Email Delivery: 1-2 minutes

**Total**: ~10-15 minutes per run

### Optimization Tips

1. **Parallel Processing**: Process multiple leads simultaneously
2. **Caching**: Cache company websites to avoid re-fetching
3. **Batch Emails**: Send all leads in one email instead of individual emails
4. **Incremental Updates**: Only fetch new information, reuse cached data

## Compliance

### GDPR Compliance

- Only collect publicly available information
- Provide unsubscribe mechanism
- Store data securely
- Delete data on request

### CAN-SPAM Compliance

- Include physical address in emails
- Provide clear unsubscribe option
- Honor opt-out requests within 10 days
- Don't use deceptive subject lines

### Best Practices

- Respect robots.txt
- Add delays between requests
- Don't overwhelm target servers
- Include privacy notice in emails
- Provide opt-out mechanism

## Troubleshooting

### Cron Job Not Running

```bash
# Check cron job status
openclaw cron list

# View cron logs
openclaw logs --filter cron

# Manually trigger the job
openclaw cron run daily-lead-gen
```

### Emails Not Sending

```bash
# Check AgentMail configuration
openclaw skills list | grep agentmail

# Test AgentMail
python3 -c "import agentmail; print('AgentMail OK')"

# Check email logs
tail -f ~/.openclaw/logs/email.log
```

### No Leads Generated

```bash
# Check if agent-browser is working
agent-browser --version

# Test search functionality
agent-browser open https://google.com
agent-browser snapshot -i

# Check search query
# Try broader or different search terms
```
