This integration automatically sends discovered leads via email using AgentMail. Each lead is sent as a formatted email with company summary and profile information.

## Implementation

### Using AgentMail Skill

```python
import os
import json
from datetime import datetime

def send_lead_email(lead_data):
    """Send a formatted lead email via AgentMail"""
    
    # Format email subject
    subject = f"New Lead: {lead_data['company']['name']} - {lead_data['contacts'][0]['title']}"
    
    # Format email body
    body = f"""
🎯 New Lead Discovery

Company: {lead_data['company']['name']}
Website: {lead_data['company']['website']}
Industry: {lead_data['company'].get('industry', 'N/A')}
Location: {lead_data['company'].get('location', 'N/A')}
Company Size: {lead_data['company'].get('size', 'N/A')}

---

Contact Information:
Name: {lead_data['contacts'][0]['name']}
Title: {lead_data['contacts'][0]['title']}
Email: {lead_data['contacts'][0].get('email', 'N/A')}
LinkedIn: {lead_data['contacts'][0].get('linkedin', 'N/A')}
Phone: {lead_data['contacts'][0].get('phone', 'N/A')}

---

Company Summary:
{lead_data.get('company_summary', 'No summary available')}

Profile Summary:
{lead_data.get('profile_summary', 'No profile summary available')}

---

Source: {lead_data.get('source_url', 'N/A')}
Extracted: {lead_data.get('extracted_at', datetime.now().isoformat())}
Status: {lead_data.get('status', 'Complete')}

---
This lead was automatically generated by OpenClaw Lead-Gen-Pro
"""
    
    # Send via AgentMail
    # Use the message tool with channel="agentmail"
    return {
        "to": "noctil.agent@agentmail.to",
        "subject": subject,
        "body": body
    }
```

## Error Handling

```python
def send_lead_with_retry(lead_data, max_retries=3):
    """Send lead email with retry logic"""
    for attempt in range(max_retries):
        try:
            result = send_lead_email(lead_data)
            if result['success']:
                return True
        except Exception as e:
            if attempt == max_retries - 1:
                # Log error and mark lead as failed
                log_error(f"Failed to send lead after {max_retries} attempts: {e}")
                return False
            # Wait before retry
            time.sleep(2 ** attempt)  # Exponential backoff
    return False
```

def is_duplicate_lead(lead_data, sent_leads_db):
    """Check if lead was already sent"""
    company_domain = extract_domain(lead_data['company']['website'])
    contact_email = lead_data['contacts'][0].get('email', '')
    
    for sent_lead in sent_leads_db['sent_leads']:
        if (sent_lead['company_domain'] == company_domain or 
            sent_lead['contact_email'] == contact_email):
            return True
    return False

## Rate Limiting

```python
import time

def send_leads_with_rate_limit(leads, delay_seconds=5):
    """Send multiple leads with rate limiting"""
    for lead in leads:
        send_lead_email(lead)
        time.sleep(delay_seconds)  # Avoid overwhelming email server
```

## Best Practices

1. **Personalization**: Include company-specific insights in summaries
2. **Timing**: Send emails during business hours in recipient's timezone
3. **Deduplication**: Track sent leads to avoid duplicates
4. **Rate Limiting**: Space out emails to avoid spam filters
5. **Error Handling**: Log failures and retry with backoff
6. **Monitoring**: Track open rates and responses
7. **Compliance**: Include unsubscribe option and privacy notice
